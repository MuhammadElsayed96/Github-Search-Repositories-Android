Github-Search-Repositories application
===================================
Simple app that allow the user to search Github repositories by a certain name, and the app will return the repositories that holds that name the user entered.

Getting Started
---------------

This sample uses the Gradle build system. To build this project, use the
"gradlew build" command or use "Import Project" in Android Studio.
